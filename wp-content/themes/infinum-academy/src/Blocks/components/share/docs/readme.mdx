# Share Component

Component that allows user to share the current article on social networks. Currently Facebook, Twitter, LinkedIn and Piterest are supported.

## Dependencies

None

## Attributes

* _@param_ **blockClass** | `string` | default: `"share"` | Component's class. If modifying make sure the provide the default class as well (see example call).

## Example call

```php
use Eightshift_Libs\Helpers\Components;

Components::render( 'share' ) );

// or

Components::render( 'share', [
	'blockClass' => 'share',
] );
```

## Implementation

Open a terminal and type in this command inside your projects root:

```shell
wp boilerplate use_component --name=share
```
